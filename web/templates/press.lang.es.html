1 de octubre de 2011

Nace el primer servicio web para gestionar enlaces a páginas web favoritas pensado y hecho en España. Skæda permite gestionar los enlaces favoritos y los visualiza con una imagen en miniatura. Los enlaces se pueden organizar por etiquetas, grupos y grupos inteligentes.

Los enlaces registrados por el usuario son privados y están alojados en la nube disponibles desde cualquier navegador web con acceso a Internet pero se pueden compartir fácilmente en las redes sociales. Importa todos tus favoritos del navegador de una vez o añádelos a tu cuenta mientras navegas con nuestro bookmarklet.

Skæda esta hecho con HTML5 y utiliza la innovadora API WebSocket que permite sincronizar en tiempo real diferentes sesiones a la vez. La aplicación funciona sin refrescar o recargar nunca la URL de trabajo. Se ha hecho un desarrollo exhaustivo con JavaScript que ha dotado a la aplicación de una funcionalidad similar a la de las aplicaciones de escritorio convencionales como la técnica de arrastrar y soltar o la búsqueda instantánea.
