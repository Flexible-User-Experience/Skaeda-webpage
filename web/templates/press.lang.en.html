October 1, 2011

Borns the first HTML5 webapp to manage your favorite web links. Skæda allows you to manage bookmarks and displays a thumbnail image. The bookmarks can be organized by tags, groups and smart groups.

Links registered by the user are private and hosted in the cloud available from any browser with Internet access but can be shared on social networks easily. Import all your favorites bookmarks at a time or add them to your account while browsing with our bookmarklet.

Skæda was made with HTML5 and uses the innovative WebSocket API that lets you real-time synchronize on several sessions at once. The application works without refreshing or reloading the URL. There has been a comprehensive development with JavaScript that has given similar functionality to that of conventional desktop applications like drag and drop or instant search.
